📊 Blinkit Sales Dashboard (Excel)

🔹 Problem Statement
Analyze Blinkit sales data to track performance and identify key business insights.

🔹 Tools Used
Microsoft Excel (Pivot Tables, Charts, Slicers)

🔹 Dataset
Sales dataset containing product, location, revenue, and order details

🔹 Key Insights

Identified top-selling product categories
Analyzed sales trends across different regions
Found revenue contribution by outlet size and location
Improved visibility of business performance using KPIs

🔹 Dashboard Features

Interactive slicers for filtering data
KPI metrics (Total Sales, Average Sales, Number of Orders)
Visual charts for trend analysis
